## 📖 About the Project

**AI Career Mentor** is an AI-powered career intelligence platform designed to help students, graduates, and job seekers understand their career readiness through intelligent resume analysis.

Instead of providing only a resume score, the platform performs an in-depth evaluation of technical skills, projects, certifications, and resume quality to generate personalized career recommendations. Users receive insights into the roles that best match their profile, the skills they need to improve, and practical guidance for becoming industry-ready.

By combining resume analytics, Artificial Intelligence, and Large Language Models (LLMs), the platform delivers an end-to-end career guidance experience—from resume evaluation to interview preparation and professional profile optimization.

Whether you're preparing for internships, campus placements, hackathons, or full-time software engineering roles, AI Career Mentor acts as a personal AI career assistant that supports every stage of your career journey.

## ✨ Core Features

| Feature | Description |
|----------|-------------|
| 📄 **Smart Resume Analysis** | Extracts resume content and identifies technical skills, projects, certifications, and education details. |
| 🤖 **AI Career Mentor** | Interactive AI assistant that answers career-related questions and provides personalized guidance. |
| 🎯 **Job Role Recommendation** | Matches resumes against multiple technology roles using intelligent weighted skill matching. |
| 📊 **Career Readiness Score** | Calculates an overall readiness score with detailed performance insights. |
| 📋 **Resume Audit** | Reviews resume quality, ATS compatibility, formatting, projects, and technical strengths. |
| 💼 **Job Description Matcher** | Compares resume content with job descriptions and highlights missing keywords. |
| 🛣️ **Career Roadmap** | Generates a personalized learning path to achieve target career goals. |
| 📚 **Course Recommendations** | Suggests relevant learning resources to bridge identified skill gaps. |
| 🎤 **Interview Preparation** | Generates technical and behavioral interview questions for selected roles. |
| 🔗 **LinkedIn Profile Optimizer** | Recommends improvements to strengthen professional online presence. |
| 💰 **Salary Insights** | Displays estimated salary ranges across different experience levels. |
| 📈 **Skill Gap Analysis** | Identifies missing skills and recommends technologies to learn next. |
